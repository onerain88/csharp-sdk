Member Function Documentation

static T LeanCloud.Utilities.Conversion.As< T > ( object  value)
inlinestatic

Converts a value to the requested type – coercing primitives to the desired type, wrapping lists and dictionaries appropriately, or else returning null.

This should be used on any containers that might be coming from a user to normalize the collection types. Collection types coming from JSON deserialization can be safely assumed to be lists or dictionaries of objects.
